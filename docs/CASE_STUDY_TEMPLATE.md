# Case Study Template

Use this structure for every portfolio case study. Keep it concise and honest.

## Frontmatter fields (required)

- `slug`
- `title`
- `summary`
- `role`
- `date`
- `stack` (list)
- `tags` (list)
- `metrics` (list)
- `confidentiality`

## Sections (required)

1. **Context** — the real problem and why it mattered
2. **My Role** — responsibilities and collaboration
3. **Stack** — tools, platforms, methods
4. **Decisions & Tradeoffs** — key choices and rationale
5. **Outcome & Impact** — results, scale, numbers
6. **Learnings** — what you learned and future improvements

## Optional

- Media section (screenshots, diagrams, links)
- Short bulleted highlights at the top
