# Fonts

By default, fonts are loaded from Google Fonts via `next/font/google`.

If you prefer local fonts, place IBM Plex files here and switch
`src/lib/fonts.ts` to use `next/font/local` instead:

- IBMPlexSans-Regular.woff2
- IBMPlexSans-Medium.woff2
- IBMPlexSans-SemiBold.woff2
- IBMPlexMono-Regular.woff2
- IBMPlexMono-Medium.woff2
